Welcome to the Shipos Kit documentation. Here you will find all the information you need to build and deploy your SaaS application.

## What is Shipos Kit?

Shipos Kit is a modern SaaS starter kit built with Next.js 15, TypeScript, and the latest web technologies. It provides everything you need to launch your SaaS product quickly.

## Key Features

- **Authentication**: Email/password and OAuth authentication with better-auth
- **Payments**: Stripe integration with subscription management
- **Database**: PostgreSQL with Prisma ORM and type-safe queries
- **API Layer**: Type-safe APIs with ORPC
- **UI Components**: Beautiful, accessible components with Radix UI
- **Internationalization**: Multi-language support with next-intl
- **AI Integration**: OpenAI integration for AI-powered features
- **Admin Panel**: User management and role-based access control
